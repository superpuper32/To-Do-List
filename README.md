# To-Do-List
Web App for managing a list of tasks with the ability to create, edit and delete tasks.

## React + TypeScript + TailwindCSS + Vite

### Technologies used

Axios, Formik, Lodash, Scss, useImmer

### System requirements

* NodeJS >= 18
* NPM >= 6.x

### Using

frontend
```sh
npm run dev
```

backend json-server
```sh
npm run server
```
